calvalus-gen --help
Usage: calvalus-gen [COMMAND] [OPTION
       calvalus-gen [ --help | -v | --version ]

Options:

  -h, --help                            Print usage
  -i, --interval                        Time interval
  -o, --output-file-path                Location to save the generate report
  -v, --version                         Print version information and quit

Commands:
    start            Start generating with the default of 30 minutes
    Stop             Stop generating

Run 'calvalus-gen COMMAND --help' for more information on a command.
