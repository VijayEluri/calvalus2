Catalogue

Calvalus input data is being registered in a new MERCI instance available to internal (!) users at

http://calvalus-merci:8080/merci

The full mission MERIS FSG, MERIS RRG, and one year (2003) of AATSR have been registered so far. Users can log into MERCI with their Calvalus credentials.

NB. The access to quicklooks and metadata details is currently not very stable because of bugs in the Hadoop fuse implementation.

Hardware resource monitoring

Ganglia has been configured for the cluster available to users at

http://www.brockmann-consult.de/ganglia/

Automatically updated graphs show cluster load, CPU and memory usage, network load, and disk I/O rates.
